# 🔐 SafeX - Secure Backend Application

## 🚀 About the Project
SafeX is a secure backend application built using Spring Boot that provides APIs with authentication and MQTT integration.

## 🛠️ Tech Stack
- Java
- Spring Boot
- REST APIs
- MQTT
- Swagger

## ✨ Features
- Authentication system
- Secure APIs
- MQTT messaging
- Swagger documentation

## ⚙️ Run Locally
bash git clone https://github.com/sannidhi-123/safeX cd safeX mvn spring-boot:run
