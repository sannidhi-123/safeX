# 🔐 SafeX - Secure Backend Application

A secure backend application built using Spring Boot that provides REST APIs with authentication, security features, and MQTT integration. The project demonstrates scalable backend architecture and secure API design.

---

## 📌 Features

### 🔐 Security & Authentication
- User authentication and authorization  
- Secure API endpoints  
- Token-based access control (JWT/session-based)  

---

### 🌐 API Functionalities
- RESTful API architecture  
- CRUD operations for resources  
- Structured request/response handling  

---

### 📡 Messaging Integration
- MQTT-based communication  
- Real-time message handling  

---

### 📊 Documentation
- API documentation using Swagger  
- Easy testing of endpoints  

---

## 🛠️ Tech Stack

- Backend: Java, Spring Boot  
- API: REST APIs  
- Messaging: MQTT  
- Documentation: Swagger  
- Build Tool: Maven

## 🧪 Getting Started

### ✅ Prerequisites

- Java (JDK 8 or above)  
- Maven  

---

### 🚀 Setup Instructions

1. Clone the repository

bash id="z1x2c3" git clone https://github.com/sannidhi-123/safeX.git 

2. Navigate to the project

bash id="l9o8p7" cd safeX 

3. Run the application

bash id="k4j3h2" mvn spring-boot:run
